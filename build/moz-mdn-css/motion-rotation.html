    The motion-rotation CSS property defines the direction of the element while positioning along the motion path.

    If 'auto' is given to motion-rotation, the element rotates in the direction of the path defined with motion-path.

    Initial valueauto
    Applies toall elements; In SVG, it applies to container elements excluding the <defs> element and all graphics elements
    Inheritedno
    Mediavisual
    Computed valueas specified
    Animation typea number
    Canonical orderthe unique non-ambiguous order defined by the formal grammar
