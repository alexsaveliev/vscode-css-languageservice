    CSS Columns

    In This Article
    1. Reference
      1. CSS Properties
    2. Guides
    3. Specifications
    4. Browser compatibility

    CSS Columns is a module of CSS that defines a multi-column layout, allowing to express how content should flow between columns and how gaps and rules are handled.
